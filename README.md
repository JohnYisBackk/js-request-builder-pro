# 🚀 API Request Builder PRO

A modern API Request Builder built with **HTML**, **CSS**, and **Vanilla JavaScript**.

This application allows you to create and send HTTP requests directly from your browser, manage custom headers, validate JSON request bodies, and inspect formatted server responses in a clean and intuitive interface.

## ✨ Features

- 🌐 Send HTTP requests
- 📡 Support for GET, POST, PUT, PATCH and DELETE
- 📝 Custom request headers
- ➕ Add and remove headers dynamically
- ✅ Real-time JSON validation
- 🎨 Pretty JSON formatter
- 📋 Copy formatted responses
- 📊 Response status indicator
- ⏱ Request duration
- 📦 Response size calculation
- ⚠ Error handling
- 🔄 Reset request builder
- 🎯 Keyboard shortcut (Enter to send request)
- 📱 Responsive modern UI
